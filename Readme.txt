#Steps to run the program

1-Download the code from github repository 

2-Open visual studio code or any another enviroment to run the code 

3-Download python interpreter version 3.9.10 because mediapipe library work with some versions of python's interpreters 

4-Download the required libraries in required file
 
5-Run the app and Move your hand in front of the camera, changing the position and shape of your fingers to see the expected text
